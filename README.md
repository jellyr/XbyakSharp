Xbyak#
============

Abstract
------------

Xbyak([https://github.com/herumi/xbyak](https://github.com/herumi/xbyak))をC#に移植したものです。  
.NETからx86/x64の機械語を出力・実行できます。

BaseVersion
------------

Xbyak ver 3.80をベースにしています。

Difference
------------

オリジナルのXbyakで使用されているいくつかのキーワードはC#で予約語となっているため、先頭に@が付いています
 * byte
 * lock
